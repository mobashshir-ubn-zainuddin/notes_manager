Tech: React (Vite), Node.js, Express.js, MongoDB Atlas, JWT, bcrypt, REST APIs, Vercel, Render

Built a full-stack Notes Management web application with secure user authentication and authorization using JWT and bcrypt.

Implemented user-scoped CRUD operations (create, edit, delete, view notes) ensuring strict data isolation between users.

Designed RESTful APIs using Node.js and Express, with structured routing and middleware-based access control.

Integrated MongoDB Atlas for persistent storage and handled production-level issues such as IP whitelisting and connection failures.

Deployed frontend on Vercel and backend on Render, resolving real-world CORS, environment variable, and cross-origin security issues.

Refactored frontend API calls to support environment-based configuration for seamless local and production builds.
